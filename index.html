[Daisy-data](https://github.com/teo3300/Daisy-data) ## WTF, I accidentally deleted this page I could be stupid ## What is Daisy-data Daisy data is my second attempt to create a telegram bot to manage STL and GCODE for a 3D printer I bought with my friends This bot basically save 3D files on a Google Drive folder and extract some data about the print (printing time, print weigh, who asked to print etc.) and put all infos in a Google SpreadSheets table (wuv U Google API) ## Other functionality I added some functions to easily resize images to create Telegram stickers ## Differences from the old Daisy Can answer in both english and italian based on user language, use an external dictionary to store user data so it's easy to manage permissions for users Also changed API from telepot to python-telegram-bot
